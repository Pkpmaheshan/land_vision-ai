# LandVision AI

LandVision AI is an AI-powered land feasibility and business planning system designed to help land owners identify the best business opportunity for their land.

## Features

- User registration and login
- Land business plan creation
- AI-based business recommendation
- Land potential score
- Scenario comparison
- Financial feasibility report
- Cost estimation
- Income prediction
- Payback period calculation
- Visual master layout plan
- Risk analysis with solutions
- SWOT analysis
- Development phases
- Marketing plan
- Printable / PDF-style report

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run

1. Download or clone this repository.
2. Copy the project folder to `C:\xampp\htdocs\`.
3. Start Apache and MySQL using XAMPP.
4. Open phpMyAdmin.
5. Create a database and import `database.sql`.
6. Open the project in browser:

```text
http://localhost/landvision-ai
